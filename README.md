# Bra Calculator

A static Bootstrap 5 bra size calculator for UK and DE sizing systems.

## Development

The application is plain HTML, CSS, and JavaScript. Bootstrap is loaded from the official jsDelivr CDN snippet for Bootstrap 5.3.x, and the application JavaScript does not depend on jQuery.

## Testing

Run all tests:

```sh
npm test
```

Run only unit tests:

```sh
npm run test:unit
```

Run only integration tests:

```sh
npm run test:integration
```

CI runs unit and integration tests on every pull request and on pushes to `main` or `master`.
